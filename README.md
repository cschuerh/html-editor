  <h1>Die Geschichte des Internets</h1>

  <h2>Frühe Anfänge</h2>
  <p>Das Internet begann in den 1960er Jahren als Forschungsprojekt unter dem Namen ARPANET.</p>
  <img src="https://upload.wikimedia.org/wikipedia/commons/6/6e/ARPANET_map_1970.png" alt="ARPANET 1970 Map" width="400">

  <h2>Die 1980er Jahre</h2>
  <p>In den 1980er Jahren wurde das TCP/IP‑Protokoll entwickelt, das die Grundlage des heutigen Internets bildet.</p>
  <img src="https://upload.wikimedia.org/wikipedia/commons/3/32/Internet_map_1989.jpg" alt="Internet map 1989" width="400">

  <h2>Das Web entsteht</h2>
  <p>1991 stellte Tim Berners‑Lee das World Wide Web Consortium (W3C) vor. Damit konnten Inhalte vernetzt werden.</p>
  <img src="https://upload.wikimedia.org/wikipedia/commons/2/27/W3C_Icon.svg" alt="W3C Icon" width="200">
